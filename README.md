Self-hosted drone launch point maps for Midlands hill sites.

## Features
- Leaflet.js + CARTO Voyager tiles
- Verified TOAL launch points (OSM cross-checked)
- Forestry England / NT / SSSI boundary overlays
- Interactive custom point pinning with localStorage persistence
- Works offline once cached

## Data Sources
- Natural England SSSI API (ArcGIS FeatureServer)
- OpenStreetMap Overpass API
- OSM-verified coordinates (April 2026)
